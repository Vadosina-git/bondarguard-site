## 13. Telegram-бот для заявок (TODO — позже, к Vercel)

- В `.env` (на Vercel): `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` — пустые заглушки в `.env.example`.
- Процедура:
  1. `@BotFather` → создать бота → получить токен.
  2. Добавить бота в чат/канал, куда должны падать заявки.
  3. `@userinfobot` или запрос `https://api.telegram.org/bot<TOKEN>/getUpdates`
     для получения `chat_id`.
  4. Записать в переменные окружения на Vercel.
